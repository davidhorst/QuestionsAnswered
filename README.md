#Questions Answered

Very simple app that allows a user to login, post questions and answer questions posted by other users.

## Motivation

Student project at Coding Dojo created start to finish in 4 hours. 

### Installing

setup the environment then

```
bower install
npm install
node start
```

and hope for the best.

## Built With

* [Angular](https://angularjs.org/) 
* [materializecss](http://materializecss.com/)
* [express](https://expressjs.com/)
